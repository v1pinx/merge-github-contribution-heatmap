# Merged GitHub Contribution Heatmap

Generate a combined GitHub contribution heatmap for multiple accounts.

Just for fun — if you work across multiple GitHub accounts (personal, work, open source) and want to flex your actual contribution activity in one place, this is for you.

![Merged Heatmap](https://merge-github-contribution-heatmap.vercel.app/v1pinx+vipinkumar-bs)

## Usage

```
https://merge-github-contribution-heatmap.vercel.app/{username1}+{username2}
```

Embed in your GitHub profile README:

```markdown
![Merged Heatmap](https://merge-github-contribution-heatmap.vercel.app/v1pinx+vipinkumar-bs)
```
